# VS Code 使用指南（Python 学习版）

## 📱 界面布局

当你打开 VS Code 时，会看到这样的界面：

```
┌─────────────────────────────────────────┐
│ 文件菜单 编辑 查看 运行 终端 帮助         │ ← 菜单栏
├─────────────────────────────────────────┤
│ 📁 │              代码编辑区              │
│ 侧 │         (你写代码的地方)            │
│ 边 │                                     │
│ 栏 │                                     │
├─────────────────────────────────────────┤
│         终端区域（显示输出结果）         │
└─────────────────────────────────────────┘
```

### 各区域功能：

| 区域 | 用途 |
|------|------|
| **左侧边栏** | 文件浏览、搜索、代码管理等功能 |
| **中间编辑区** | 编写和编辑代码 |
| **底部终端** | 运行代码、查看输出、执行命令 |
| **右上角图标** | 快速操作按钮 |

## 📂 文件和文件夹操作

### 打开文件夹（第一次使用）

1. 点击菜单：**文件** → **打开文件夹**
2. 选择 `d:\PythonProjects\learnTest` 这个目录
3. 点击"选择文件夹"

### 创建新文件

**方法 1**（推荐）：
- 右键点击左侧文件夹 → **新建文件**
- 输入文件名（如 `list_learn.py`）

**方法 2**：
- 快捷键 `Ctrl + N` → 创建空白文件 → `Ctrl + S` 保存时指定名称

### 删除文件

- 右键文件 → **删除** 或按 `Delete` 键

## 🐍 运行 Python 代码

### 方式 1：直接运行（最简单）

1. 打开你的 Python 文件（如 `hello.py`）
2. 按 `Ctrl + F5` 运行代码
3. 看底部终端的输出结果

### 方式 2：在终端中运行

1. 打开终端：`Ctrl + `` （反引号）
2. 输入命令：
   ```powershell
   python hello.py
   ```
3. 按 Enter 执行

### 方式 3：交互式运行（选定代码）

1. 选中你要运行的代码
2. 右键 → **在终端中运行**
3. 或按 `Ctrl + Alt + N`（如果安装了 Code Runner 扩展）

## 🔧 VS Code 设置和扩展

### 推荐安装的扩展（对 Python 学习很有帮助）

1. **Python**（微软官方）
   - 在左侧扩展面板搜索 "Python"
   - 点击"安装"

2. **Pylance**（智能代码补全和错误检查）
   - 搜索 "Pylance" → 安装

3. **Code Runner**（快速运行代码）
   - 搜索 "Code Runner" → 安装

### 如何安装扩展

1. 点击左侧边栏的**扩展**图标（方块图标）
2. 在搜索框输入扩展名
3. 点击"安装"
4. 等待安装完成

## 📝 编辑代码的常用技巧

### 代码补全和提示

- 开始输入时，VS Code 会自动显示建议
- 按 `Ctrl + Space` 强制显示提示
- 按 `Enter` 或 `Tab` 接受建议

### 多行编辑

- 按住 `Ctrl` 点击多个位置，可以同时编辑多处
- 按 `Ctrl + Alt + ↑/↓` 在上下添加游标

### 缩进

- `Tab` 向右缩进
- `Shift + Tab` 向左缩进
- 选中多行后使用上面的快捷键可以批量缩进

## 🐛 调试代码

### 设置断点

1. 点击代码行号左侧，会出现红点
2. 按 `F5` 进入调试模式
3. 代码会停在断点处
4. 可以看到右侧的**变量**和**监视**窗口

### 调试控制

| 按钮 | 快捷键 | 功能 |
|------|--------|------|
| **继续** | F5 | 运行到下一个断点 |
| **单步进入** | F11 | 进入函数内部 |
| **单步跳过** | F10 | 跳过函数，执行下一行 |
| **单步返回** | Shift + F11 | 从函数中返回 |
| **停止** | Shift + F5 | 停止调试 |

## 💡 常见问题和解决方案

### 问题 1：无法运行 Python 代码

**解决方案**：
1. 确保已安装 Python（在终端输入 `python --version`）
2. 安装 Python 扩展（详见上面的"扩展"部分）

### 问题 2：代码有红色波浪线

**可能是**：
- 语法错误 → 检查代码是否有拼写错误
- 缺少模块 → 需要安装相关库

**解决方案**：
- 鼠标悬停在红线上，会显示错误提示
- 按 `Ctrl + .` 可以看到快速修复建议

### 问题 3：找不到文件/终端执行出错

**解决方案**：
1. 先在 VS Code 中打开文件夹（不是直接打开文件）
2. 终端应该自动进入该目录
3. 如果不在正确目录，手动切换：`cd d:\PythonProjects\learnTest`

### 问题 4：想清除终端输出

**快捷键**：
- 在终端输入 `cls`（Windows）或 `clear`（Mac/Linux）

## 🎯 学习 Python 时的最佳实践

1. **每学一个知识点，就创建一个新的 .py 文件练习**
   - 例如：`01_list_practice.py`、`02_loop_practice.py` 等

2. **使用注释记录学习笔记**
   ```python
   # 这是列表的基本用法
   my_list = [1, 2, 3, 4, 5]
   print(my_list[0])  # 打印第一个元素
   ```

3. **经常运行代码查看结果**
   - 这样可以立即反馈，加深理解

4. **使用调试功能**
   - 当代码结果不符合预期时，设置断点调试

5. **保存所有练习文件**
   - 方便以后回顾和复习
